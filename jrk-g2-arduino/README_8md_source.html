8 ## Summary
9 
10 This is a library for the Arduino IDE that helps interface with a
11 [Jrk G2 USB Motor Controller with Feedback][jrk]
12 using serial or I&sup2;C.
13 
14 ## Supported platforms
15 
16 This library is designed to work with the Arduino IDE versions 1.8.x or later;
17 we have not tested it with earlier versions. This library should support any
18 Arduino-compatible board, including the [Pololu A-Star controllers][a-star].
19 
20 ## Getting started
21 
22 ### Hardware
23 
24 The Jrk G2 USB Motor Controllers with Feedback can be purchased from Pololu's
25 website. Before continuing, careful reading of the [Jrk G2 User's Guide][guide]
26 is recommended.
27 
28 ### I2C connections
29 
30 To control the Jrk G2 using I2C, you should connect your board's SCL pin to the
31 Jrk's SCL pin, connect your board's SDA pin to the Jrk's SDA pin, and connect
32 one of your board's GND pins to one of the Jrk's GND pins.
33 
34 ### Serial connections
35 
36 To control the Jrk G2 using asynchronous TTL serial, you need to at least
37 connect your board's TX pin (as defined in the table below) to the Jrk's RX pin,
38 and connect your board's GND pin to one of the Jrk's GND pins. If you want to
39 read information from the Jrk, you must also connect your board's RX pin (as
40 defined in the table below) to the Jrk's TX pin.
41 
42 The example sketches for this library use a hardware serial port on your Arduino
43 if one is available: if you Arduino environment defines
44 `SERIAL_PORT_HARDWARE_OPEN`, the examples will use that port. The pins for this
45 serial port are different depending on which Arduino you are using.
46 
47 | Microcontroller Board | Hardware serial? | MCU RX pin | MCU TX pin |
48 |-----------------------|------------------|------------|------------|
49 | A-Star 32U4 | Yes | 0 | 1 |
50 | A-Star 328PB | Yes | 12 | 11 |
51 | Arduino Leonardo | Yes | 0 | 1 |
52 | Arduino Micro | Yes | 0 | 1 |
53 | Arduino Mega 2560 | Yes | 19 | 18 |
54 | Arduino Due | Yes | 19** | 18 |
55 | Arduino Uno | No | 10 | 11 |
56 | Arduino Yun | No | 10 | 11 |
57 
58 ** The Due's serial port is 3.3&nbsp;V, so it should not be directly connected
59 to the Jrk's 5&nbsp;V TX line. A voltage divider or level shifter can be
60 used.
61 
62 ### Jrk configuration
63 
64 Before using the example sketches, you should use the
65 Jrk G2 Configuration Utility software to apply these settings:
66 
67 * Control mode: Serial/I&sup2;C/USB.
68 * Serial mode: UART, fixed baud rate
69 * Baud rate: 9600
70 * CRC disabled
71 * Device number: 11
72 
73 ### Software
74 
75 You can use the Library Manager to install this library:
76 
77 1. In the Arduino IDE, open the "Sketch" menu, select "Include Library", then
78  "Manage Libraries...".
79 2. Search for "JrkG2".
80 3. Click the JrkG2 entry in the list.
81 4. Click "Install".
82 
83 If this does not work, you can manually install the library:
84 
85 1. Download the
86  [latest release archive from GitHub](https://github.com/pololu/jrk-g2-arduino/releases)
87  and decompress it.
88 2. Rename the folder "jrk-g2-arduino-xxxx" to "JrkG2".
89 3. Drag the "JrkG2" folder into the "libraries" directory inside your
90  Arduino sketchbook directory. You can view your sketchbook location by
91  opening the "File" menu and selecting "Preferences" in the Arduino IDE. If
92  there is not already a "libraries" folder in that location, you should make
93  the folder yourself.
94 4. After installing the library, restart the Arduino IDE.

96 ## Examples
97 
98 Several example sketches are available that show how to use the library. You can
99 access them from the Arduino IDE by opening the "File" menu, selecting
100 "Examples", and then selecting "JrkG2". If you cannot find these
101 examples, the library was probably installed incorrectly and you should retry
102 the installation instructions above.
