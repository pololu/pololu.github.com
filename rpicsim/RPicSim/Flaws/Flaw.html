Overview

Represents a flaw in RPicSim, usually due to bugs or limitation of the MPLAB X classes we are using. Stores the name of the flaw and knowledge about what versions of MPLAB X it affects and how.

def effect(version)
  if @versions.key? version
    @versions[version]
  else
    @probable_affect_for_other_versions
  end
end
