7 ## Summary
8 
9 This is a C++ library for the Arduino IDE that helps access the on-board hardware of the [A-Star 32U4 controllers](http://www.pololu.com/category/149), which is a family of programmable controllers from Pololu based on the ATmega32U4. You can program an A-Star 32U4 directly from the Arduino IDE without using this library, but this library makes it easy to:
10 
11 * display information on the LCD
12 * play sounds and music on the buzzer
13 * read the battery voltage level
14 * control the three user LEDs
15 * use the three user pushbuttons
16 * use the on-board motor drivers
17 
18 (Not all of these features are available on every A-Star 32U4. For more information about each particular A-Star controller, please refer to their documentation and respective product pages.)

20 ## Installing the library
21 
22 If you are using version 1.6.2 or later of the [Arduino software (IDE)](http://www.arduino.cc/en/Main/Software), you can use the Library Manager to install this library:
23 
24 1. In the Arduino IDE, open the "Sketch" menu, select "Include Library", then "Manage Libraries...".
25 2. Search for "AStar32U4".
26 3. Click the AStar32U4 entry in the list.
27 4. Click "Install".
28 
29 If this does not work, you can manually install the library:
30 
31 1. Download the [latest release archive from GitHub](https://github.com/pololu/a-star-32u4-arduino-library) and decompress it.
32 2. Rename the folder "a-star-32u4-arduino-library-master" to "AStar32U4".
33 3. Move the "AStar32U4" folder into the "libraries" directory inside your Arduino sketchbook directory. You can view your sketchbook location by opening the "File" menu and selecting "Preferences" in the Arduino IDE. If there is not already a "libraries" folder in that location, you should make the folder yourself.
34 4. After installing the library, restart the Arduino IDE.

57 ## Component libraries
58 
59 This library also includes copies of several other Arduino libraries inside it which are used to help implement the classes and functions above:
60 
61 * [FastGPIO](https://github.com/pololu/fastgpio-arduino)
62 * [PololuBuzzer](https://github.com/pololu/pololu-buzzer-arduino)
63 * [PololuHD44780](https://github.com/pololu/pololu-hd44780-arduino)
64 * [Pushbutton](https://github.com/pololu/pushbutton-arduino)
65 * [USBPause](https://github.com/pololu/usb-pause-arduino)
66 
67 The Pushbutton and FastGPIO libraries are generally useful and you might want to use them directly in your sketch. You can use these libraries automatically without any extra installation steps and without needing to add any extra `#include` lines to your sketch.
68 
69 You should avoid adding extra `#include` lines such as `#include <Pushbutton.h>` because then the Arduino IDE might try to use the standalone %Pushbutton library (if you previously installed it), and it would conflict with the copy of the Pushbutton code included in the AStar32U4 library. The only `#include` line needed to access all features of this library is:
70 
71 ~~~{.cpp}
72 #include <AStar32U4.h>
73 ~~~
